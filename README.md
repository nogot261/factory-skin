# Учебный сайт по варианту М: производство изделий из кожи

Фамилия: Матвеева. Первая буква фамилии - М. Предметная область по заданию: производство изделий из кожи.

## Состав

- PHP-сайт без CMS.
- 8 основных страниц: главная, о компании, каталог, производство, уход, новости, контакты, карта сайта.
- Дополнительные страницы: поиск, вход, личный кабинет, 404.
- Два режима оформления: стандартный и версия для слабовидящих.
- Роли пользователей: администратор, менеджер, клиент.
- Форма обратной связи сохраняет обращения в `storage/messages.json`.

## Тестовые учетные записи

- `admin` / `admin2025` - администратор
- `manager` / `manager2025` - менеджер
- `client` / `client2025` - клиент

## Запуск в XAMPP

1. Установить XAMPP.
2. Скопировать папку `leather_factory_site` в каталог `htdocs`.
3. Запустить Apache в панели XAMPP.
4. Открыть: `http://localhost/leather_factory_site/index.php`.

## Размещение на хостинге

Подойдет любой PHP-хостинг с поддержкой PHP 8.x и записи в каталог `storage`.

GitHub Pages не подходит, потому что сайт написан на PHP: он использует серверные страницы, сессии и запись формы обратной связи в `storage/messages.json`. GitHub Pages отдает только статические HTML/CSS/JS-файлы и не выполняет PHP-код.

## Деплой на Render

Для Render подготовлен `Dockerfile` на базе `php:8.3-apache`.

Порядок деплоя:

1. Создать на Render новый **Web Service**.
2. Подключить репозиторий `nogot261/factory-skin`.
3. В качестве окружения выбрать **Docker**.
4. Dockerfile path оставить `./Dockerfile`.
5. После деплоя открыть выданную Render ссылку.

Важно: на бесплатном тарифе Render файловая система контейнера не предназначена для постоянного хранения данных. Сообщения из формы могут пропасть после пересборки или рестарта. Для учебной демонстрации этого достаточно, для постоянного проекта нужна база данных или внешнее хранилище.
